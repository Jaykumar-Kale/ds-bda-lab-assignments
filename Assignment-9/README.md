# Assignment 9 – Ecommerce Review Scraper

## AIM
To create a review scraper for an ecommerce website using Python.

## OBJECTIVES
- Understand Web Scraping
- Extract real-time customer reviews
- Store scraped data into CSV format

## THEORY

Web scraping is the automated extraction of data from websites.

Steps involved:
1. Send HTTP request
2. Retrieve HTML content
3. Parse HTML structure
4. Extract required data
5. Store data in structured format

Libraries used:
- requests → Send HTTP request
- BeautifulSoup → Parse HTML
- pandas → Store structured data

## IMPLEMENTATION

The scraper extracts:
- Customer Name
- Rating
- Review Title
- Review Text

It supports multi-page scraping.

The extracted data is stored in reviews.csv.

## CONCLUSION

Web scraping was successfully implemented using Python.

Customer reviews were extracted and stored in CSV format for analysis.
